Plan: Implement 10-Point Scoring in Music Recommender
Context
Point-weighting scheme for song recommendations:

genre: 4 pts (exact categorical match)
mood: 3 pts (exact categorical match)
energy: 2 pts (continuous, proximity-based)
valence: 1 pt (continuous, proximity-based)
Max score = 10. Wired into both OOP (Recommender) and functional (recommend_songs) paths. UserProfile.likes_acoustic → target_valence.

Scoring Formula

score = 0
if song.genre == user.favorite_genre:  score += 4
if song.mood  == user.favorite_mood:   score += 3
score += 2 * (1 - abs(user.target_energy  - song.energy))   # 0–2
score += 1 * (1 - abs(user.target_valence - song.valence))  # 0–1
# max = 10
